Overall, this project used R to analyze the relationship between atmospheric CO₂ levels and global temperature anomalies. Using the tidyverse package, the CO₂ data from the Mauna Loa Observatory and the global temperature data from NASA were imported, cleaned, and merged into a single dataset based on month. After preparing the data, correlation analysis and visualizations were created using ggplot2 to examine the relationship between CO₂ concentration and global temperature. A scatterplot with a fitted linear regression line showed a strong positive association between the two variables. Residual plots were also used to check the assumptions of linear regression, including linearity, constant variance, and approximate normality of residuals. A linear regression model was then fitted in R to quantify the relationship, and a confidence interval and hypothesis test for the slope were calculated. The extremely small p value and the confidence interval excluding zero provided strong statistical evidence that increases in CO₂ levels are associated with increases in global temperature anomalies. Although the model has limitations, such as using global averages and not accounting for other climate influences, the analysis demonstrates a clear positive relationship between atmospheric CO₂ concentrations and global temperature change.
